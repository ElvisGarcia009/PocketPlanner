**PocketPlanner**

PocketPlanner es una aplicación móvil diseñada para la gestión integral de las finanzas personales. La aplicación permite a los usuarios registrar y categorizar ingresos y gastos, ajustar presupuestos personalizados mediante inteligencia artificial y recibir asesoría financiera tipo chatbot para optimizar sus hábitos de gasto y ahorro.

Características

**Registro y Autenticación**  
Permite a los usuarios crear una cuenta y acceder de forma segura a la aplicación. Puede iniciar con google o registrar una cuenta PocketPlanner con autenticación de email.

**Gestión de Transacciones** 
Los usuarios pueden registrar y categorizar manualmente sus ingresos y gastos, o importar información desde correos electrónicos.

**Presupuestos Personalizados** 
La aplicación utiliza un sistema de tarjetas o card donde el usuario puede crear, editar o borrar un presupuesto; dentro del presupuesto puede agregar items a través de un sistema de iconos que representan los gastos de dicho presupuesto.

**Presupuestos con IA** 
Utilizando algoritmos RandomForestRegressor la IA realiza predicciones de cuanto terminará gastando por item el usuario cuando llegue la quincena/mensualidad y, a través de reglas heurísticas, propone montos nuevos para los items que excedan o no cumplan con el presupuesto.

**Asesoría Financiera con IA** 
Utilizando la API de OpenAI tenemos un chatbot inteligente que se encarga de leer todos tus movimientos registrados en la aplicación durante tu periodo de nómina (ya sea quincenal o mensual) y ofrece asesoría financiera sobre el estado actual del usuario con respecto a sus ingresos/gastos. El usuario puede realizar cualquier pregunta siempre y cuando tenga que ver con sus finanzas. Tambien se puede utilizar el chatbot como guía para saber como funciona la aplicación y como utilizarla.

**Metas de Ahorro** 
Permite establecer objetivos financieros y proyectar el progreso hacia dichos objetivos.

**Notificaciones y Alertas** 
Envío de alertas en tiempo real sobre gastos y límites presupuestarios.
